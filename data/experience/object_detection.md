# Real-Time Object Detection and Monitoring System

**Type:** Personal project
**Domain:** Computer vision

**What I built:**
A real-time object-detection API using YOLOv8 and FastAPI, with WebSocket support for
live video-stream inference, containerized with Docker Compose. I also developed a
Streamlit dashboard for live detection visualization, bounding-box overlays, and
historical performance analytics with alerting.

**Tools & tech:** YOLOv8, FastAPI, WebSockets, Docker Compose, Streamlit

**Details:**
- Live video-stream inference over WebSockets.
- Bounding-box overlay visualization.
- Historical performance analytics with alerting.
