# Triumph Capital — Machine Learning Engineer & Data Analyst

**Role:** Machine Learning Engineer and Data Analyst
**Location:** Surat, GJ, India
**Dates:** Aug 2024 – Jul 2025
**Domain:** Finance / quantitative equity research

**What I did:**
Developed and automated stock-scanning algorithms in Python using pandas and NumPy,
integrating statistical indicators (Relative Strength, ADX) and a backtesting
framework. Built end-to-end data pipelines that ingested over 5 million rows of
market data via yfinance, cleansed the data with vectorized pandas/NumPy operations,
and loaded it into SQL Server. Implemented batch scheduling and rate-limit controls
that improved data freshness from daily to hourly. Designed and deployed a Power BI
dashboard coupled with a Streamlit front end that let portfolio managers filter stocks
against custom statistical thresholds.

**Tools & tech:** Python, pandas, NumPy, yfinance, SQL Server, Streamlit, Power BI,
backtesting framework, statistical indicators (Relative Strength, ADX)

**Metrics / impact:**
- Cut manual stock-screening time by ~70% through automation.
- Improved market-data freshness from daily to hourly via batch scheduling + rate-limit controls.
- Reduced portfolio-manager decision latency by ~40% with the dashboard.
